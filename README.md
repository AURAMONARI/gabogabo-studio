# 🏋️‍♂️ Gabogabo Studio - Gestione Recensioni

Sito web completo e dinamico sviluppato in Python con il framework Flask, progettato per gestire le recensioni degli utenti con supporto al caricamento di immagini e memorizzazione persistente dei dati.

## 🚀 Link al Sito Pubblicato
🔗 **[Visualizza il sito live qui](gabogabo-studio.vercel.app)**

---

## 🛠️ Tecnologie Utilizzate

* **Backend:** Python 3, Flask
* **Database:** SQLite3 (Database relazionale SQL)
* **Frontend:** HTML5, CSS3 (Layout moderno con effetti responsive)
* **Controllo Versione:** Git & GitHub

---

## 📦 Caratteristiche Principali

* **Lettura Dinamica (SELECT):** Mostra le recensioni dei clienti direttamente dal database SQL in card grafiche avanzate.
* **Inserimento Nuovi Dati (INSERT INTO):** Form di invio recensioni completo (Nome, Servizio, Voto da 1 a 5, Commento).
* **Gestione Immagini:** Supporto per il caricamento di foto e salvataggio sul server.
* **Area Admin Protetta:** Sistema di eliminazione delle recensioni tramite password di sicurezza (`hihihiha`).
* **Inizializzazione Automatica:** Il database si autoconfigura al primo avvio se il file non è presente.
